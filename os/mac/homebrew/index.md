## 使用
```
# 查询可用包
$ brew search <pkg>
# 查看包信息
$ brew info <pkg>
# 安装包
# 例：$ brew install node@12
$ brew install <pkg>
# 删除包
$ brew uninstall <pkg>
# 设置链接
# 例：brew link --overwrite -f node@12
$ brew link <pkg@n>
# 删除链接
$ brew unlink <pkg@n>
# 已安装列表
$ brew list
# 帮助信息
$ brew -h
# 查看Homebrew版本
$ brew -v
# 更新Homebrew
$ brew update
# 体检
$ brew doctor
```
