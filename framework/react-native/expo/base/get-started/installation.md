# [安装]

## 1. 本地开发工具：Expo CLI
```
# 1. git 安装
$ git --version
# 2. node 安装（最新稳定）
$ brew update
$ brew info node
$ brew install node
$ node -v
$ npm i -g npm
# 3. 安装 expo-cli
$ npm i -g expo-cli
$ npm i -g sharp-cli
# 4. 安装 watchman(mac)
$ brew install watchman
```
